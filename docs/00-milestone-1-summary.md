## Technology Stack Explained

### Frontend: Next.js + Tailwind CSS
**What it means:** Modern, fast website framework
**Benefits:**
- Lightning-fast page loads
- Excellent SEO performance
- Beautiful, responsive design
- Future-proof technology

### Backend: WordPress (Headless)
**What it means:** WordPress for content management only (no WordPress theme)
**Benefits:**
- Familiar interface for content updates
- Easy to add publications and blog posts
- Secure and reliable
- No technical knowledge needed

### How They Work Together
```
You add content in WordPress → Next.js fetches it → Visitors see beautiful, fast website
```
