## Project Stack

- **Frontend:** Next.js 14+ with App Router
- **Styling:** Tailwind CSS
- **CMS:** WordPress (Headless)
- **Deployment:** Vercel (frontend) + WordPress hosting
- **API:** WordPress REST API or WPGraphQL
