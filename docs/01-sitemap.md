# Africa CSID Website Sitemap

## Website Structure

### 1. Home
- Hero section with mission statement
- Impact metrics (At a Glance)
- Featured programs overview
- Latest news/blog highlights
- Call-to-action buttons (Partner With Us, Get Involved)
- Partners/Platforms logos

### 2. About Us
**Main Page:** Organization overview, vision, mission, values

**Sub-pages:**
- **Who We Are** - Detailed organizational background, approach (voice, capacity, influence)
- **Our Team** - Team members with photos, roles, bios (WordPress managed)
- **Platforms & Partnerships** - Collaborative platforms (AFSTC, JRT, ISFAA, CSA MSP, KCCWG)

### 3. What We Do (Programs)
**Main Page:** Overview of three core pillars

**Sub-pages:**
- **Peace & Conflict Transformation** - Detailed program information, approach, impact
- **Economic Empowerment** - Skills development, income diversification initiatives
- **Climate Change** - Climate resilience, adaptation, emission reduction programs

Each program page includes:
- Program description
- Objectives
- Approach/methodology
- Impact stories
- Related publications
- Related news articles

### 4. Resources
**Sub-pages:**
- **Publications** - Downloadable reports, policy briefs, research (WordPress managed)
  - Filterable by topic, year, program area
  - PDF downloads
  - Featured publications section
  
- **News & Blogs** - Articles, updates, thought leadership (WordPress managed)
  - Category filtering
  - Search functionality
  - Featured posts
  
- **Events** - Past and upcoming events (WordPress managed)
  - Calendar view
  - Event details pages
  - Registration links (if applicable)
  
- **Gallery** - Photos and videos from field work
  - Organized by program/location
  - Lightbox viewing

### 5. Get Involved
**Sub-pages:**
- **Partner With Us** - Partnership opportunities, contact form
- **Work With Us** - Current vacancies, application process (WordPress managed)

### 6. Contact
- Contact form

- Social media links
- Map integration (optional)

## URL Structure

```
/                           → Home
/about                      → About Us main
/about/who-we-are          → Who We Are
/about/team                → Our Team
/about/platforms           → Platforms & Partnerships

/programs                   → Programs overview
/programs/peace-conflict   → Peace & Conflict Transformation
/programs/economic         → Economic Empowerment
/programs/climate          → Climate Change

/resources/publications    → Publications listing
/resources/publication/[slug] → Individual publication
/resources/blogs           → Blog listing
/resources/blog/[slug]     → Individual blog post
/resources/events          → Events listing
/resources/event/[slug]    → Individual event
/resources/gallery         → Photo/Video gallery

/get-involved              → Get Involved main
/get-involved/partner      → Partner With Us
/get-involved/careers      → Work With Us

/contact                   → Contact page
```
