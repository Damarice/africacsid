## 3. Confirmation of Tools, Platform, and Approach

### Frontend:
- **Next.js** – Server-side rendering, static generation, optimal SEO
- **Tailwind CSS** – Fast, responsive, modern styling

### Backend:
- **WordPress (Headless CMS)** – Manages content (publications, blogs, events, team) without interfering with frontend

### Hosting:
- **Frontend** hosted on Vercel (fast, CDN-enabled)
- **WordPress** hosted on existing hosting

### Approach:
1. Build fast, responsive, modern frontend with React + Tailwind
2. Use WordPress purely for content management (simple for client use)
3. Implement SEO best practices: clean URLs, metadata, sitemaps
4. Design reusable components and layouts for future scalability
5. Provide full handover with documentation and training

### Workflow:
```
You add content in WordPress → Next.js fetches it → Visitors see a beautiful, fast website
```
